# popular-packages.json

Hand-curated list of popular npm packages used for typosquatting detection via Levenshtein distance.

**Source:** [Socket.dev popular packages](https://socket.dev/npm/category/popular) (pages 1-5, retrieved 2026-04-07)

**Maintenance:** This list requires manual updating. To refresh, review the Socket.dev popular category and update the JSON file directly. Keep it sorted alphabetically.
