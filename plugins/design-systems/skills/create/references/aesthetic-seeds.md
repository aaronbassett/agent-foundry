# Aesthetic Seeds

> Non-default starting directions for design system generation.
> Each seed breaks the model out of statistical convergence by
> providing opinionated font pairings, color strategies, and
> spatial philosophies that are NOT the training data median.

## How Seeds Work

The create skill selects the most appropriate seed based on the
user's description. The seed is a starting direction, not a template.
The generated design system adapts and builds on the seed's foundation.

## Banned Fonts

No seed may use: Inter, Roboto, Open Sans, Lato, Arial.
These are the statistical defaults of AI training data.

## Editorial

### The Broadsheet — Editorial

**Character**: Authoritative, spacious, and warm — the feeling of opening a well-set broadsheet.

**Font Pairings**:
- Display: Fraunces (700-900)
- Body: Source Serif 4 (300-400)
- Mono/Data: JetBrains Mono

**Color Strategy**: Derive from newsprint — warm creams (#faf5e8), charcoal (#2d2926), one ink-red accent (#c41e3a).

**Spatial Philosophy**: Generous whitespace, asymmetric 2-column, wide margins.

**Elevation**: No shadows — tonal shifts between cream layers only.

**Natural Prohibitions**: No gradients, no dark mode, no rounded corners over 4px.

**Font Pairings**:

## Brutalist

### The Manifesto — Brutalist

**Character**: Unapologetic, loud, and structurally honest — the anti-decoration manifesto.

**Font Pairings**:
- Display: Epilogue (800-900)
- Body: Work Sans (400)
- Mono/Data: Space Mono

**Color Strategy**: Flat — pure neutrals (#1a1a1a, #f5f5f0, #e0e0dc) with one high-voltage accent (#ff3300).

**Spatial Philosophy**: Strict visible grid, 0px radius everywhere, heavy 4px borders.

**Elevation**: Hard offset shadows (8px 8px 0px #000).

**Natural Prohibitions**: No gradients, no blur, no rounded corners, no opacity below 100%.

**Font Pairings**:
- Display: Space Grotesk (700)
- Body: DM Sans (400)

## Minimal

### The Void — Minimal

**Character**: Nearly invisible — the interface dissolves until only the content remains.

**Font Pairings**:
- Display: Geist (200-700)
- Body: Geist (300-400)
- Mono/Data: Geist Mono

**Color Strategy**: Near-monochrome — one background (#fafafa or #111111), one text (#222222 or #eeeeee), one accent (any single saturated color).

**Spatial Philosophy**: Maximum whitespace, single column, one element per viewport section.

**Elevation**: None — all flat, borders at 5% opacity if needed.

**Natural Prohibitions**: No more than 3 colors total, no shadows, no gradients, no hover effects beyond cursor change.

## Maximalist

### The Trading Desk — Maximalist

**Character**: Overwhelming by design — every pixel earns its place through information density.

**Font Pairings**:
- Display: Clash Display (600-700)
- Body: Plus Jakarta Sans (400-500)
- Mono/Data: Fira Code

**Color Strategy**: Dark base (#0c0c14), multi-accent system — each data category gets its own color (green for positive, red for negative, blue for neutral, amber for warning).

**Spatial Philosophy**: Tight grid, no wasted space, data tables as primary content, sidebar + main + panel layout.

**Elevation**: Subtle 1px borders between panels, tonal surface shifts for depth.

**Natural Prohibitions**: No whitespace larger than 1.5rem, no serif fonts, no centered layouts.

## Atmospheric

**Character**: Cinematic and intimate — the interface recedes into darkness so the content can glow.

**Font Pairings**:
- Display: Sora (600-800)
- Body: Karla (300-400)
- Mono/Data: Azeret Mono

**Color Strategy**: Derive from analog photography — deep darkroom black (#0a0a0a), safe-light red (#8b0000 at 40% for accents), silver halide (#c0c0c0 for text), amber (#d4a847 for highlights).

### The Aurora — Atmospheric

**Character**: Vast, shifting, and luminous — an interface that feels like standing under a polar sky.

**Font Pairings**:
- Display: Newsreader (400-700)
- Body: DM Sans (300-400)
- Mono/Data: Fira Code

**Color Strategy**: Derive from polar light — deep navy (#0b132b), teal (#3a506b), aurora green (#5bc0be), ice white (#c9e4e7).

**Spatial Philosophy**: Flowing full-width sections, overlapping transparent layers, content floats on color fields.

**Elevation**: Glassmorphism used intentionally (not as default) — frosted panels over gradient backgrounds, 20px blur.

**Natural Prohibitions**: No flat/solid backgrounds for hero sections, no monospace outside data contexts, no hard borders.
